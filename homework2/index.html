連結設定

  • 一般連結 : 連結
  • 改變顏色 : color
  • 滑動顏色 : hover
  • 點擊動作 : active

這章節你認識的CSS設定文字語法

  1. 文字顏色 : color
  2. 文字大小 : font-size
  3. 文字行距 : line-height
  4. 文字字型 : font-family
  5. 文字中線 : text-decoration line-through
  6. 文字底線 : text-decoration underline
  7. 其他 : ...

下列的emmet語法，哪一項是錯的?

  • ! = HTML5環境
  • link:src=匯入CSS
  • p*4 = 顯示4個p段落
  • h1 = 展開h1的開頭與結尾html標籤
  • lorem = 自動就幫我們寫完網頁產生假字補空白文章內容
